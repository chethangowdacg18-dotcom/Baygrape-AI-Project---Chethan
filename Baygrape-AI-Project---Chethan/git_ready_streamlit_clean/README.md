# Streamlit RAG (Single App)

## Run
1. Create and activate a virtual environment.
2. Install dependencies:
   pip install -r requirements.txt
3. Start app:
   streamlit run app.py

## Notes
- Single Streamlit app (no FastAPI backend needed).
- Chroma persists locally in `./chroma_store`.
- Page backgrounds are loaded from `./background_assets`.
